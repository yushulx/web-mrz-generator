# MRZ Generator - Chrome Extension

Generate Machine Readable Zone (MRZ) codes for travel documents directly in your browser.

https://github.com/user-attachments/assets/d34cdca7-2968-46a3-a32d-0fbb2a17303a

## Features

✅ **5 Document Types Supported:**
- TD3 - Passport
- TD2 - Travel Document
- TD1 - ID Card
- MRVA - Visa Type A
- MRVB - Visa Type B

✅ **Quick Generation:**
- Generate MRZ codes instantly
- Random data generator for testing
- Real-time preview

✅ **Export Options:**
- Download as PNG image
- Save for testing and mockups

✅ **Privacy First:**
- 100% local processing
- No data collection
- No internet connection required
- No tracking or analytics

✅ **Realistic Output:**
- ICAO-compliant MRZ codes
- Authentic-looking document design
- Proper check digit calculation
- Professional formatting

## How to Use

1. Click the extension icon in your Chrome toolbar
2. Select document type from the dropdown
3. Fill in the required information, or click "Random Data" for testing
4. Click "Generate MRZ" to create the document
5. Download the image or copy to clipboard

## Use Cases

- **Developers**: Test MRZ scanning applications

- **Designers**: Generate mockup travel documents

- ICAO 9303 compliant
